## 功能介绍

挂机修仙游戏，提供了 身世，开图，修炼，挂机，炼丹，练药等功能。

使用技术 vue，cocoscreator3.8.0，nodejs，express，mysql，redis，腾讯cos存储

## 使用教程（必须）

1. 下载后有 3 个项目包，分别是 admin ， RestService ，ImmortalRoad ，他们分别是后台管理系统，服务器，游戏本体

2. admin是一个前端项目，使用的框架为vue2，RestService是基于express的nodejs项目，ImmortalRoad 则是cocos游戏项目。

3. 游戏使用的数据库为 mysql 和 redis 请提前安装准备好，所接入的存储服务为腾讯的cos存储。

4. admin 项目直接 npm install 后 npm run serve 就可以启动

5. ImmortalRoad 项目直接用 cocoscreator 3.8.0 导入即可

6. RestService需要修改目录中的.env 文件替换为自己的配置

6. 根目录下存在预设 database.sql 文件可以直接导入数据库使用

   .env文件配置详解如下

   ```
   SERVER_PORT=4567 #服务器启动端口
   
   WEBSOCKET_PORT=4568 # websocket端口
   
   REDIS_HOST=127.0.0.1 # redis主机
   REDIS_PORT=6379 # redis端口
   
   MYSQL_PORT=3306 # mysql端口
   MYSQL_ACCOUNT=root # mysql账号
   MYSQL_PASSWORD=123456 # mysql密码
   MYSQL_DATABASE=immortal_road # mysql数据库名称
   
   TOKEN_KEY=asfda4961v-dfv4d1s65g4-dg41sd6g54fds-vbc16b4fd98 # token加密密钥这个随便写一个就行
   
   # 邮箱配置
   # 示例
   # EMAIL_ACCOUNT=2124564@qq.com
   # EMAIL_PASS=54f5as4f5as1fa56s4f
   # 详情参考 https://www.bilibili.com/video/BV16z4y1D74s
   EMAIL_ACCOUNT=??? # smtp服务邮箱账号支持qq，网易，Google等邮箱 
   EMAIL_PASS=??? # smtp服务邮箱密码
   
   # 三方腾讯云存储配置 
   # 详情参考 https://www.bilibili.com/video/BV1F14y1e7Fd
   Region=??? # 存储桶所在地域
   BUCKET_NAME=??? # 存储桶名称
   SECRITE_ID=??? # 腾讯云存储桶id
   SECRITE_KEY=??? # 腾讯云存储桶key
   ```
